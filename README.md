# Intella Hiring Challenges

Welcome to the Intella hiring challenges repository! This repository contains technical assessments for various job roles at Intella.

## About This Repository

This is a collection of assignments designed to evaluate candidates for different positions at Intella. Each folder contains a specific role-based challenge with its own detailed requirements and instructions.

## For Candidates

### Getting Started

1. **Fork this repository**
   - Click the "Fork" button at the top right of this repository
   - You can fork it as either a public or private repository
   - If private, make sure to add Intella reviewers as contributors

2. **Navigate to your role-specific folder**
   - Each folder (e.g., `backend/`, `frontend/`, etc.) contains a specific challenge
   - Read the `README.md` inside your assigned folder carefully

3. **Complete the assignment**
   - Follow the instructions in the role-specific README
   - Implement the required features and functionality

4. **Code quality matters**
   - Write clean, maintainable code
   - Follow best practices for your technology stack
   - Include proper documentation

### Notes on AI Tools

Using AI assistants (GitHub Copilot, Claude, ChatGPT, etc.) is **encouraged**. We're interested in:

- Your ability to architect a solution
- How you read, interpret, and modify generated code
- Your critical evaluation of AI suggestions
- The final quality of the deliverable

### Submission

1. Push your final code to your repository (public or private)
2. Ensure the README contains all instructions needed to run your project
3. **Add Intella reviewers as contributors** to your repository (usernames will be provided)
4. Reply to your contact at Intella to confirm submission
